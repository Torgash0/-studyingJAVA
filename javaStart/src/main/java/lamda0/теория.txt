В Java символ " -> " используется для определения лямбда-выражений. Лямбда-выражения в Java представляют собой анонимные
 функции, которые могут быть переданы в качестве параметров или сохранены в переменных.

Синтаксис лямбда-выражения в Java выглядит следующим образом:
(parameters) -> expression
где "parameters" - список параметров, "expression" - тело функции, которое может содержать одно или несколько выражений.

Например, следующий код создает лямбда-выражение, которое принимает два аргумента типа int и возвращает их сумму:
(int a, int b) -> a + b

Лямбда-выражения в Java обычно используются вместе с функциональными интерфейсами, которые имеют только один абстрактный метод.
Это позволяет использовать лямбда-выражения для реализации этого метода без необходимости создавать отдельный класс или интерфейс.
