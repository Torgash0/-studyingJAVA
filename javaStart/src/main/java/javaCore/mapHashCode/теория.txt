КАРТЫ
Map<K, V> (key, value)
Hashap (наиболее популярная)
хранит ключи в хеш таблице
имеет наиболее высокую производительность
не гарантирует пордяок элементов
может содержать null ключи и  null занчения

TreeMap
хранит ключи в отсортированном порядке
работает медленнее
не может содержать null ключи , но может содержать null занчения

LinkedHashMap
хранит ключи в порядке их вставки в Map
немного медленнее HashMap
 может содержать null ключи и  null значения
